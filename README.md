# React-Material-ui-Template
## Used
1. [React](https://reactjs.org/)
2. [NEXT.js](https://nextjs.org/)
3. [Material-ui](https://material-ui.com/)
4. [React-grid-layout](https://github.com/STRML/react-grid-layout)
5. [Material-table](https://material-table.com/)
6. [Echarts](https://github.com/apache/incubator-echarts)
7. [Echarts-for-react](https://github.com/hustcc/echarts-for-react)

## Install
```
yarn install
yarn dev

http://localhost:3000/view-name
```
